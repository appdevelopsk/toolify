# 被リンク・配信プラン(#4) — 正直版

> ⚠️ ここは**私(Claude)が自動実行できない**領域です。被リンクは外部サイト/人の判断で生まれるもので、スパム的な投下はBAN+逆効果。
> 私ができるのは「リンクされやすい資産の用意」と「正当な配信先リスト+下書き」まで。最終的な投稿・申請は運営者が行う(またはアカウントを渡して私が代行)。

## 大原則
- **量産リンクは厳禁**(ディレクトリ一括登録・コメントスパム・相互リンク集 = 今のHCU/スパム評価をさらに悪化させる)。
- 狙うのは **少数の文脈に合った自然リンク**。妊娠・妊活クラスタは「共有したくなる独自性(不規則周期対応・6周期一覧・.ics・検診スケジュール日付化)」があるので、そこを訴求する。

## A. リンクされやすくする資産(私が用意できる / 一部実装済み)
1. **クラスタの独自機能そのもの**(済): 他ツールに無い「検診スケジュール日付化」「不規則周期レンジ」「IVF/超音波対応の予定日」=引用・紹介の理由になる。
2. **埋め込み(embed)**(既存 `/embed/...` あり): 「あなたのブログに無料で貼れる排卵日計算機」として配布 → 設置=被リンク。embed の存在を各ツールページで告知する導線を足すと効く(未実装・候補)。
3. **OG画像**(動的生成あり): SNS共有時の見栄え=クリック率。クラスタ専用のOG文言は確認推奨。

## B. 正当な配信先(運営者が実行 / 下書きは私が作成可)
| チャネル | 妥当性 | 注意 |
|---|---|---|
| 妊活・出産系の**個人ブログ/掲示板の文脈内紹介** | 高 | 宣伝臭を消し「この計算機が便利」という体。質問への回答内で1リンク |
| **Reddit** r/TryingForABaby, r/BabyBumps 等 | 中 | self-promo規制が厳しい。価値ある回答の中で稀に。連投BAN |
| **Pinterest**(検診スケジュール/妊娠週数の図) | 中 | 画像主体。クラスタの図解と相性可 |
| **はてな/note/個人ブログ**(運営者自身の解説記事→ツール導線) | 高 | 自分の資産からの内部的リンク。安全 |
| ニッチな**まとめ/ツール紹介系**への純粋な掲載依頼 | 中 | 相手の編集基準次第。一括登録系はNG |

repo の `scripts/` に既存の SNS 投稿スクリプト(reddit/hn/buffer)があるが、**ツールリンクの機械投下には使わない**(前述のBANリスク)。

## C. 推奨アクション(順序)
1. まず **7/1 の計測**を待つ。クラスタに表示/順位が戻り始めているか確認してから配信投資する(死んでいる段階で配信しても無駄+リスク)。
2. 回復の兆しが出たら、**運営者自身のブログ/note に妊活ジャーニー解説記事**を書き、その中からクラスタ各ツールへリンク(最も安全で文脈的)。
3. embed 配布の導線をツールページに追加(私が実装可)。
4. 各クラスタツールに「このツールを紹介/引用してください」的な引用しやすい一文+正規URLを置く(bar="cite this tool")。

## 私が次にできる具体実装(指示あれば)
- [ ] 各ツールページに **embed コード提示UI**(コピー用 iframe スニペット)を追加 → 設置=被リンク導線
- [ ] クラスタ用の **「引用・共有」ブロック**(正規URL + 一文)を追加
- [ ] 運営者ブログ用の**解説記事の下書き**(妊活→出産、各ツールへ自然リンク)を作成

> 結論: 被リンクは「待つ→自分の資産から張る→共有しやすくする」が安全な順。外部への機械投下はしない。
